How it works

gvNIX permite construir el modelo de datos de una aplicación mediante ingeniería directa e inversa y permite generar una aplicación de gestión (web o no) en múltiples tecnologías de visualización (Spring MVC, Flex, JSF, Vaddin..)

gvNIX aprovecha las características de Spring Roo y aporta su propia filosofía: ofrecer al desarrollador componentes de alto valor funcional para aplicaciones corporativas.
