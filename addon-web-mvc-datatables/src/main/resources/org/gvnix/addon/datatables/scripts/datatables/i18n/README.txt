The contained in this folder are Datatables label translations for many languages. 

Each file has been created from the contents of the following page: 
http://www.datatables.net/plug-ins/i18n

To add a new locale, create a file using the following naming template: 
labels_LANGUAGE.txt

For the LANGUAGE, use the two letter language identification, as used in 
the java Locale object. Language variants and the java Locale policy is 
not supported (ex: no en_UK vs. en_US).
